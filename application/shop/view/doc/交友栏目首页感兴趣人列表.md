
## 交友栏目首页，感兴趣的人列表

功能：

1. 显示我关注的人的视频列表。

~~~
/user/friend/comrade_list
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|用户id|
|sort_scheme|排序参数|
|startid|起始序号|
|length|长度|

| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| list     | array | 视频列表，含义参见[交友栏目首页](/shop/doc/index/name/交友栏目首页) |
| is_bottom     | int | 1到底。 |
