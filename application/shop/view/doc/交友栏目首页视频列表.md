功能：

1. 显示我关注的人的视频列表。

~~~
/user/friend/video_list
~~~
~~~

~~~

| -------- |:------|

| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| list     | array | 我关注的人的视频列表，含义见《交友栏目首页》。 |
| is_bottom     | int | 1到底 |
