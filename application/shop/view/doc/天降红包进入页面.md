
## 天降红包进入页面

功能：

1. 用户进入页面，我这里给一些检查。
1. 查uid
1. 查token
1. 查时间段是否合适
1. 查用户的波币是否超过20波币。
1. 查用户当天是否已玩过，每天只能一次，以点击play按钮为准。


~~~
/game/moneyrain/index
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|当前用户id|
|token|当前用户token|


| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| 无     |  |  |
