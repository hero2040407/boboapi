
## 导师栏目首页的视频列表

~~~
/record/starmaker_v3/record_list
~~~
~~~
GET
~~~


功能：导师栏目的新首页的精彩视频列表，分页展示。2018 03  
  

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|startid  |  起始序号|
|length |  长度 |

|返回字段型 |类型 | 含义 |
| -------- |:------|:------|
|   list   | array | 精彩视频的列表，详细字段这里不写，同其他视频一致。 |
|   is_bottom   | int | 1到底 |
