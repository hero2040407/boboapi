## 打赏视频接口_波豆

功能：打赏视频，使用波豆，老的打赏视频接口请不要再使用。  
功能：ip校验，打赏人和被打赏人如果处于同一ip，提示“违规操作”  
功能：手机绑定校验。  

~~~
/shop/dashangbean/index
~~~
~~~
POST
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|  打赏人用户id，是付钱的人|
|room_id| 短视频或直播，或回播的room_id，不是主键id  |
|type| 礼物主键，参加 《礼物列表》  |

| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| current_gold       | int | 付钱后，当前用户的 波币 数 |
| bean               | int | 被打赏人得到的 波豆 数 |


| 错误代码        |含义  |
| -------- |:------|
| -202      |波币余额不足  |
