
## 拉黑检查
~~~
/user/relation/lahei_check
~~~
~~~
GET
~~~

功能：检查一个人是否拉黑另一个人

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid                |  用户id|
|target_uid         |  目标用户|


返回： 
| 字段        | 类型 |含义  |
| -------- |:------|:------|
|check_result   | bool    | 为真表示拉黑。 |
