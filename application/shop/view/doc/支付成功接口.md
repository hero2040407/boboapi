## 查订单支付状态接口

功能：服务端接到请求后，向支付宝或微信服务器发请求，得到状态就返回给调用此接口的客户端。  
*注意：服务端不对订单做任何处理，本接口仅查询。*   
*注意：本接口适用范围：充值和购物均可，仅限现金支付用。且仅查询一次第三方服务器。*  
*注意：本接口无视当前数据库订单状态，直接发送远程请求。*

~~~
/shop/orderquery/index
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|order|  商城订单号|


| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| success | int |1:支付成功，0未支付成功  |
