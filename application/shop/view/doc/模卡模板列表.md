
## 模卡模板列表

功能：展示所有的模板，一般来说，一个模板有多张样例图片。

~~~
/user/card/template
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无|   |

| 返回字段        | 类型  | 含义  |
| -------- |:------|:------|
|  list | array  | 模板数组，见下 |


| list每行        | 类型  | 含义  |
| -------- |:------|:------|
|  id | int  | 模板id，非常重要，制作模卡时，传给服务器 |
|  title | string  | 模板标题 |
|  pic_arr | array  | 样例图片数组，安卓用 |
|  pic_width_height_arr | array  | 样例图片数组,ios用，见下 |

| list每行        | 类型  | 含义  |
| -------- |:------|:------|
|  pic | string  | 图片 |
|  pic_width | int  | 图片宽 |
|  pic_height | int  | 图片高|
