
## 用户VIP表单申请

功能：

1. 用户在不满足6个特定条件下申请vip，付费之后，需要填写这个表单。




~~~
/user/vip/apply
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|用户id|
|token|用户token|
|name|用户自己填写的自己真实姓名|
|phone|用户自己填写的真实手机号，客户端需自行通过验证码的方式确认手机号真实性|
|info|用户自己填写的特长，必填。|



| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| 无     |  |  |
