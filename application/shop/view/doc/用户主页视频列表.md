## 用户主页视频列表

功能：返回某个用户的主页视频列表，含短视频和回播，分页显示。

~~~
/user/info/get_user_video
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid | 个人主页uid，可能不是当前用户的  |
|self_uid | 手机当前用户uid  |
|startid | 起始序号  |
|length | 长度  |



| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| list         | array   | 视频列表，每个视频的字段参见[视频列表新接口](/shop/doc/index2/name/视频列表新接口) |
| is_bottom         | int   | 1到底  |
