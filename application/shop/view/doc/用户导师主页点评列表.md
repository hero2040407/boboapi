## 用户导师主页点评列表

功能：返回某个导师用户的点评列表，分页显示。  
功能：只有导师主页，才应该调用此接口

~~~
/user/info/tutor_comment_list
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid | 个人主页uid，这个人必须是导师  |
|self_uid | 当前用户  |
|startid | 起始序号  |
|length | 长度  |



| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| list         | array   | 视频列表，每个视频的字段参见[视频列表新接口](/shop/doc/index2/name/视频列表新接口) |
| is_bottom         | int   | 1到底  |
