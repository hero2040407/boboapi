
## 用户微信网页登录

功能：

1. 用户在微信公众号里，用菜单调用app的接口。  
1. 注意，其实此接口由微信服务器调用，web网页只需填写这个接口即可。
1. 但是，因为使用了客户端重定向，本接口的返回会返回给网页端。
1. **本接口的返回等同于app的登录和注册接口**

~~~
/user/login/weixin_index
~~~
~~~
POST
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
| code     | 微信生成的code |

返回：本接口的返回等同于app的登录和注册接口
