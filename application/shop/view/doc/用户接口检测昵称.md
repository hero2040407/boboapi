
## 用户修改检测昵称

功能：

1. 用户修改检测昵称，如当月已申请过，则返回消息，否则code为1表示用户可以修改昵称（也只是申请）

~~~
/user/nickname/check
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|用户id|


| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| 无     |  |  |

返回示例
~~~
code:0,
message: 您在2017-09-01日已修改过昵称
~~~
