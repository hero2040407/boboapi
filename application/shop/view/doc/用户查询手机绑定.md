
## 用户查询手机绑定

功能：

1. 主要提供给node调用。
1. 查询用户是否有手机绑定，如果有一个手机注册的账号，那么返回真，如果其他方式注册的帐号，再绑定手机注册，也返回真。。

~~~
/user/check/is_bind_phone
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|用户id|



| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| success     | int | 1表示该用户用手机方式注册过或绑定过，0表示没有 |
| new_role_name     | string | sign:签约用户，vip：vip用户，normal:普通用户 |
