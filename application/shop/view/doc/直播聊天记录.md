## 直播聊天日志
~~~
/push/chat/save_info
~~~

功能:node调用，记录聊天信息

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid         |  说话人用户id|
|room_id     |  房间id|
|info         | 聊天内容|


返回： 
| 字段        | 类型 |含义  |
| -------- |:------|:------|
|无   |     |  |
