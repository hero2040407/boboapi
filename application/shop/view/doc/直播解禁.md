## 直播解禁
~~~
/user/jubao/cancel_zhibo
~~~
~~~
POST
~~~

功能：直播解禁，由后台调用
会通过nodejs发送消息如下。

     $data =[
            'uid' => $uid,
            'message' =>  "您的直播屏蔽已经解除",
     ];


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid         |  待解禁用户id|


返回： 
| 字段        | 类型 |含义  |
| -------- |:------|:------|
|无   |     |  |
