
## 视频删除回播

~~~
/push/rewindmanager/remove_rewind
~~~
~~~
POST
~~~


功能：用户删除自己的回播。  

注意：这是假删除，只是修改了字段。



| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|短视频作者uid|
|id|回播视频的主键id，不是room_id |
|token | 安卓大于3.2.3，苹果大于3.2.0，增加此参数，校验身份，就是登录时返回的token |

| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| 无     |  | |
