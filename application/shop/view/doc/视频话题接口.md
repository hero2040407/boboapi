
## 视频话题接口

~~~
/record/theme/index
~~~
~~~
GET
~~~


功能：上传短视频前，用此接口获取已有话题。


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid| 当前用户uid |




| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| list_hot     | array |最热话题列表，返回个数服务器定 |
| list_recent | array |  个人最新的话题列表，返回个数服务器定,每行的字段同 list_hot |

| list_hot每行        | 类型 |含义  |
| -------- |:------|:------|
| id     | int |话题id，其实对客户端无用 |
| theme_title     | string |话题内容 |
