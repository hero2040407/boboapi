
## 视频邀请点评，导师偏好设置

~~~
/record/starmaker_v2/set_preference
~~~
~~~
POST
~~~


功能：导师设置自己想看的视频的分类  
功能：设置之后，进入导师点评主页时，点评广场接口，和官方推送接口，需更新。  
功能：默认是全选的，这由服务端设置好，客户端不需管这个。  

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid  |  用户id|
|token  |  token|
|ids  |  类似1,2，1和2是导师偏好获取接口我返回的，注意，至少得选一个偏好|



|返回字段型 |类型 | 含义 |
| -------- |:------|:------|
|   无   |  |  |
