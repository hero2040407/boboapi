## 购买视频接口

功能：扣减用户波币，并记录，以后该用户观看该视频再也不用付费。  目前只有短视频和回播要购买。
~~~
/shop/video/buy
~~~
~~~
POST
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid|  用户id|
|room_id| 视频room_id） |


| 返回字段        | 类型 |含义  |
| -------- |:------|:------|
| current_gold | int | 购买后用户剩余波币  |

| 错误代码        |含义  |
| -------- |:------|
| -202      |波币余额不足  |
