
## 二级页面童星排行


~~~
/api/vip/index
~~~
~~~
GET
~~~


功能：  

1. 获取二级页面的童星排行，是分页的。  


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid       |  用户id，如不传按10000处理|
|startid       |  起始序号|
|length       | 长度 |
|type       | 1魅力颜值，2活跃人气，如不传默认1. |


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| is_bottom |  int   | 1到底， |
| list |  array   | 新闻列表，可能包含大赛。区分方法，根据bigtype字段，如果是race表示大赛，如果是news表示新闻 |
| rank |  int   | 请求参数uid在排行榜的位置。注意，每个人在两种type的排行榜的位置是不同的。 |


| list 每行        | 类型 |含义  |
| -------- |:------|:------|
| uid |  int   | uid |
| pic |  string   | 用户图片 |
| nickname |  string   | 昵称 |
| badge |  string   | 小图标 |
