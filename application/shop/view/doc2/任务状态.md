## 任务状态

功能：查看任务状态。  
~~~
/task/taskapi/get_taskstate
~~~
| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid         |  用户id|
|task_id     |  任务主键，注意不是活动主键，应该对于任务列表中的id |

返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| complete | string    | 1完成，0未完成 |
| reward   | string    | 0不可以领奖，1可以领奖但未领，2已领奖 |
