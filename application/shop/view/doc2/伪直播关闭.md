
## 伪直播关闭

功能：关闭一个伪直播  ，应该仅被node调用。
功能：查询条件是，直播正在播放，且类型为伪直播，如满足则关闭。

~~~
/push/chat/sham_push_end
~~~
~~~
POST
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
| uid         | 想关闭的伪直播主播uid|



返回：   无论什么情况都返回正常
| 返回        | 类型 |含义  |
| -------- |:------|:------|
| 无      |   |   |
