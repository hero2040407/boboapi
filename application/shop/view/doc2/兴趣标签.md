## 兴趣标签接口

功能：返回所有的兴趣id，图片，名称。
~~~
/api/boboapi/get_label2018
~~~
~~~
GET
~~~


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无         | |



返回：   
| 返回        | 类型 |含义  |
| -------- |:------|:------|
| list     | array    | 全部兴趣的数组， |


| list每行        | 类型 |含义  |
| -------- |:------|:------|
| id      | int  | id，是唯一的|
| name    | string  | 名称，例如跳舞|
| num     | int    | 人数，暂未用 |
| image   | string    | 普通的图片地址 |
| image2018  | string    | 上传短视频时，用的特定图片 |
