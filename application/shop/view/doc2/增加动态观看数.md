
## 增加动态观看数

功能：加一些动态观看的记录

1. 动态都是至少1条。
1. 随机挑选  2 个类型为10,11的用户（批量导入的用户）作为机器人
1. 随机挑选  record_count 个动态（概率 1周内 50%   一月内40%  一个月前10%）
1. 对于每条动态，每人插入浏览记录（概率100%），每个人的浏览次数（1 -100随机）
1. 添加评论（概率3%）

~~~
/api/robotupdates/index
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|record_count         |  修改的动态条数|


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| 无 |     |  |
