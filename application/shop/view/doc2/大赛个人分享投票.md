
## 大赛个人分享投票



~~~
/api/share/like
~~~
~~~
GET
~~~
~~~
v=5
~~~

功能：  

1. 给分享页面投票，是大赛的。


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|self_uid       |  当前用户uid|
|token       |  当前用户token|
|uid       |  被投票的uid|
|race_id       |  大赛id|
|type       |  1普通投票，2分享投票，3波币购买投票|



返回：      
| 字段        | 类型|参数含义  |
| -------- |:------|:------|
|无   |     |  |
