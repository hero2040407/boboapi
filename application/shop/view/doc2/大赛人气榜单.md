
## 大赛人气榜单

~~~
/race/share/rank
~~~
~~~
GET
~~~
~~~
v=5
~~~

功能：  

1. 大赛人气榜单


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|race_id       |  大赛id|
|startid       |  起始id|
|length       |  长度|
|v       |  5|
|search       |  搜索的uid |


返回：      
| 字段        | 类型|请求参数含义  |
| -------- |:------|:------|
|list   | array    |  人气榜单，字段见下|

| list 每行        | 类型|请求参数含义  |
| -------- |:------|:------|
|uid   | int    | uid|
|pic   | string    | 头像 |
|name   | string    | 姓名 |
|ticket_count   | int    | 票数 |
