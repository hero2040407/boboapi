## 大赛单个详情h5

功能：  

1. h5页面专用，返回大赛或者活动的详情



~~~
/race/index/h5_detail
~~~


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|id       |  大赛id，或者活动id|
|type       |  race表示大赛，act表示活动|


| 返回字段        | 类型|含义  |
| -------- |:------||:------|
|detail   | string      |  大赛或活动的详情，是富文本页面，|
| css_content          | string    | css文件的内容 |
