
# 官方问答

## 热门问题

~~~
/api/question/hot
~~~
~~~
GET
~~~

功能：  

1. 列出热门问题，最多5个。

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无         | |

返回：   

| 返回字段        | 类型|含义  |
| -------- |:------|:------|
|list    |  array   |  问题列表，见下 |

| list每行        | 类型|含义  |
| -------- |:------|:------|
|id    |  int   |  id |
|title    |  string   |  标题 |
|answer    |  string   |  内容  |
|type    |  int   |  1充值问题，2直播问题，3账号问题，4封号查询，5其他问题 |


<br><br><br>



## 问题列表（按分类）

~~~
/api/question/index
~~~
~~~
GET
~~~

功能：  

1. 按分类列出问题列表，无分页

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|type         | 1充值问题，2直播问题，3账号问题，4封号查询，5其他问题|

返回：   

| 返回字段        | 类型|含义  |
| -------- |:------|:------|
|list    |  array   |  同上|

<br><br><br>
