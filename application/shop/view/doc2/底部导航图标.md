
## 底部导航图标（安卓版本）

~~~
/api/boboapi/bottom_bar_for_android
~~~
~~~
GET
~~~


功能：  

1.  客户端通过此接口显示底部导航的图标，因为这些图标经常变动。
1.  这是安卓的。  

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无         |  |


| 返回 | 类型       | 请求参数含义  |
| -------- |:------|:------|
|object     | object |其实是一个数组，但是有键 |

| object | 类型       | 请求参数含义  |
| -------- |:------|:------|
| show    | object |格式见下 |
| live    | object |格式见下 |
| my    | object |格式见下 |
| friend    | object |格式见下 |
| activity    | object |格式见下 |


| list每个值的统一格式 | 类型       | 请求参数含义  |
| -------- |:------|:------|
| color_2x_url    | string |对应图片网址 |
| color_3x_url    | string |对应图片网址 |
| gray_2x_url    | string |对应图片网址 |
| gray_3x_url    | string |对应图片网址 |



## 底部导航图标（苹果版本）

~~~
/api/boboapi/bottom_bar_for_ios
~~~
~~~
GET
~~~


功能：  

1.  客户端通过此接口显示底部导航的图标，因为这些图标经常变动。
1.  这是苹果的。  
1.  只要版本号服务端和客户端请求的不一致，则客户端就需要下载zip文件，一样则不需要。注意：服务端版本号是有可能小于客户端的。  


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|version         | 导航图标的版本号，客户端目前的，客户端最开始可以传0 |


| 返回 | 类型       | 请求参数含义  |
| -------- |:------|:------|
|zip     | string |一个网址，内含该版本所有的底部导航图片。 |
|version     | int |  导航图标的版本号，服务器当前使用的  |
|list     | object |其实是一个数组，但是有键，格式见上方的安卓版，唯一差别是，每个url仅仅是一个文件名，无路径无域名，且一定被含在zip文件里。 |
