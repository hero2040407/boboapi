## 波币使用日志

功能：分页显示当前用户波币使用日志，支出和收入都显示。负数代表支出  
*注意：这里只显示波币相关，不显示波豆。*
~~~
/user/bobi/index
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid         |  用户id|
|startid     | 起始序号|
|length      | 长度|



返回：   
| 返回        | 类型 |含义  |
| -------- |:------|:------|
| result | array | 见下 |
| is_bottom | int | 1到底，0没有 |

| result每行        | 类型 |含义  |
| -------- |:------|:------|
| way      | string  | 文字说明|
|  count    | int    | 波币使用，负数表示支出 |
| time  | int    | 当时的时间，时间戳 |
