
## 用户权限检查


~~~
/api/auth/check
~~~
~~~
GET
~~~


功能：  

1. 检查各个操作的权限，
1. 如果完全不允许用户做当前操作，返回code=0，message为提示信息。


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid       |  用户id|
|type       | 必传，1动态发布检查， |


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| status |  int   | 根据请求type的不同，含义不同 |


| 请求 type | 返回 status        | 类型 |含义  |
| -------- |:------|:------|
| 1 |  1   | int|显示直播按钮 |
| 1 |  2   | int|显示模卡按钮，不显示直播按钮。 |
| 1 |  3   | int|显示灰色的不能点击的模卡按钮。 |
| 1 |  4   | int|显示4个全部的按钮，动态图片，动态视频，模卡，直播4个按钮。 |
