
## 短视频生成

功能：node调用，添加一条短视频  
功能：node端最好以shell方式调用，而不是以http请求调用。  
功能：node端尽量使用单独程序调用，以方便修改请求 的时间和频率。  

~~~
/systemmanage/recordfake/index
~~~
| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无         | |



返回：   
| 返回        | 类型 |含义  |
| -------- |:------|:------|
| record_id      |  int |  新生成的短视频id |
