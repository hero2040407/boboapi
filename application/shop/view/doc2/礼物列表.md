## 礼物列表

功能：返回所有礼物，目前按照价格从小到大。
~~~
/api/present/index
~~~
~~~
GET
~~~

| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|         |  |


返回：   
| 数组的每行        | 类型 |含义  |
| -------- |:------|:------|
| id          | int | 礼物主键，重要 |
| title       | string    | 礼物名称 |
| pic         | string | 礼物图片 |
| gold        | int    | 价格多少波币 |
| experience  | int    | 加多少经验值 |
