
## 试镜卡绑定


~~~
/advise/card/bind
~~~
~~~
POST
~~~


功能：  

1. 绑定试镜卡，只能是线下销售的。     
1. 很多情况不能绑定，    
1. -- 号码填写错误  
1. -- 已经参加过当前通告。   


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|uid       |  用户id，|
|token       |  token|
|serial       | 用户购买的试镜卡的序列号，不分大小写。 |
|advise_id       | 通告id |
|role_id       | 角色id |


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| 无 |     |  |
