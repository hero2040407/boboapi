
## 轮播图新通用接口


~~~
/api/toppic/newindx
~~~
~~~
GET
~~~


功能：  

1. 轮播图通用接口。


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|module_name       |  必传，star_top表示星动态栏目的，index_top,表示app首页,brandshop:品牌馆栏目共通，audition_card:试镜卡栏目共通|


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| list |  array   | 图片数组，见下 |


| list 每行       | 类型 |含义  |
| -------- |:------|:------|
| id |  int   | id |
| picpath |  string   | 图片自身网址 |
| linkurl |  string   | 跳转网址 |
