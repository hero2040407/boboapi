## 通告类型接口


~~~
/api/advise/type_list
~~~
~~~
GET
~~~


功能：  

1. 给出通告的类型，便于客户端筛选时显示。


| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|无       |  |


返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| id |  int   | 类型的主键 |
| name |  string   | 名称 |
