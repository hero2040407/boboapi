

## 邀约单个活动信息h5

功能：返回邀约单个活动的信息,只返回html格式的详情。

~~~
/task/taskactivityapi/h5_detail
~~~
~~~
GET
~~~



| 请求参数字段        | 请求参数含义  |
| -------- |:------|
|id         |  活动id|

返回：   
| 字段        | 类型 |含义  |
| -------- |:------|:------|
| detail          | string    | h5详情 |
| css_content          | string    | css文件的内容 |
